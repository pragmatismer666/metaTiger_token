# Meta Tiger Token

This project contains source code of token Meta TIGER (TIGR)

node v14.18.2
npm 6.14.15
hardhat 2.7.0

To deploy clone repo then, create a `.env` file and write

```shell
PRIVATE_KEY = "YOUR_PRIVATE_KEY"
``` 

Then run

```shell
npx hardhat run scripts/deploy.js --network mainnet
```
